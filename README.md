# University ERP System

A Spring Boot application for managing university electives, timetables, and student/faculty dashboards.

## Features
- Role-based login (Student, Faculty, Admin)
- Timetable management
- Elective selection with validation
- Academic handbook integration

## Tech Stack
- **Backend**: Java, Spring Boot, JPA/Hibernate, H2 Database
- **Frontend**: HTML5, CSS3, JavaScript (Vanilla)
- **Build Tool**: Maven

## Getting Started
1. Clone the repository.
2. Run `./mvnw spring-boot:run` to start the application.
3. Access the portal at `http://localhost:8080/login.html`.
