Detailed Syllabus Section

CS3223: Deep Neural Networks

L-T-P-C: 3-0-0 | Credits: 3

Prerequisites: Basic Python, Linear Algebra, Calculus, ML fundamentals

Unit 1: Fundamentals of Artificial Neural Networks

Introduction to biological neurons vs artificial neurons. Perceptron learning rule, convergence theorem. Multi-layer Perceptron (MLP), Activation functions (Sigmoid, ReLU, Tanh). Backpropagation algorithm and Mathematical derivation.

Unit 2: Optimization and Regularization

Gradient Descent variants (Stochastic, Mini-batch). Hyperparameter tuning. Regularization techniques: L1, L2, Dropout, Batch Normalization. Initialization strategies (Xavier, He).

Unit 3: Convolutional Neural Networks (CNN)

Spatial patterns and Convolutional layers. Pooling layers. Popular architectures: AlexNet, VGG, ResNet, Inception. Applications in Image Classification and Object Detection.

Unit 4: Recurrent Neural Networks (RNN)

Sequence modeling. Vanishing and Exploding gradients. LSTM (Long Short-Term Memory) and GRU (Gated Recurrent Units). Bidirectional RNNs. Applications in Time-series and Language modeling.

Unit 5: Advanced Topics and Generative Models

Autoencoders and Denoising. Introduction to Generative Adversarial Networks (GANs). Attention mechanism and Transformer basics. Case studies in Computer Vision.

Total Hours: 45

Textbooks
  • "Deep Learning" by Ian Goodfellow, Yoshua Bengio, and Aaron Courville.
  • "Neural Networks and Deep Learning" by Michael Nielsen.
Reference Books
  • "Hands-On Machine Learning with Scikit-Learn, Keras, and TensorFlow" by Aurélien Géron.
  • "Deep Learning with Python" by François Chollet.

CS2202: Computer Networks

L-T-P-C: 3-0-0-3 | Credits: 3 | Faculty: Dr. Tauheed

Prerequisites: Data Structures, Operating Systems basics

Unit 1: Introduction & Physical Layer

Data communication concepts, network topologies, OSI vs TCP/IP model. Transmission media: guided (twisted pair, coaxial, fiber optic) and unguided. Signal encoding, bandwidth, data rate, Nyquist and Shannon theorems.

Unit 2: Data Link Layer

Framing, error detection (CRC, Checksum, Hamming), error correction. Flow control: Stop-and-Wait, Sliding Window. MAC sublayer: ALOHA, CSMA/CD, CSMA/CA. Ethernet, 802.11 (Wi-Fi) standards.

Unit 3: Network Layer

IPv4 addressing and subnetting, CIDR. Routing algorithms: Distance Vector (RIP), Link State (OSPF), BGP. IPv6 addressing and transition. NAT, ICMP, ARP, RARP.

Unit 4: Transport Layer

TCP vs UDP. TCP connection establishment (3-way handshake), flow control (windowing), congestion control (Slow Start, Congestion Avoidance, Fast Retransmit). Port numbers and socket programming.

Unit 5: Application Layer & Security

DNS, HTTP/HTTPS, FTP, SMTP, POP3, DHCP, SNMP. Network security basics: symmetric/asymmetric encryption, TLS/SSL, firewalls, VPN, intrusion detection.

Total Hours: 45

  • "Data Communications and Networking" by Behrouz Forouzan

Unit 5: Compliance & Ethical Hacking

Security standards: ISO 27001, NIST, GDPR. Penetration testing lifecycle. Common vulnerabilities: SQL injection, XSS, CSRF. Incident response procedures and digital forensics overview.

CS4122: Reinforcement Learning

L-T-P-C: 3-0-0-3 | Credits: 3 | Faculty: Dr. Gourav

Prerequisites: Python, Probability & Statistics, Linear Algebra, Basic ML

Unit 1: Foundations of RL

Agent-Environment interaction. Markov Decision Processes (MDPs): states, actions, rewards, policy, value functions. Bellman equations. Exploration vs Exploitation dilemma (epsilon-greedy).

Unit 2: Dynamic Programming

Policy Evaluation, Policy Improvement, Policy Iteration. Value Iteration algorithm. Convergence guarantees. Generalized Policy Iteration (GPI). Applications to finite MDPs.

Unit 3: Model-Free Methods

Monte Carlo methods: prediction and control. Temporal Difference (TD) learning: TD(0), SARSA, Q-Learning. TD vs Monte Carlo tradeoffs. n-step TD methods.

Unit 4: Function Approximation & Deep RL

Linear and nonlinear function approximation. Deep Q-Network (DQN), experience replay, target networks. Policy Gradient methods: REINFORCE. Actor-Critic architectures (A3C, PPO).

Unit 5: Advanced Topics & Applications

Multi-agent reinforcement learning. Inverse RL and imitation learning. RL in robotics, game playing (AlphaGo, OpenAI Five), and autonomous vehicles. Ethical considerations in RL deployment.

Total Hours: 42

CS4178: Modern Cryptography

L-T-P-C: 3-0-0-3 | Credits: 3 | Faculty: Dr. Ravi Kishore

Prerequisites: Discrete Mathematics, Number Theory basics, Algorithms

Unit 1: Mathematical Foundations

Modular arithmetic, Euler's theorem, Fermat's little theorem. Groups, rings, fields. Primitive roots and discrete logarithm. Quadratic residues. Chinese Remainder Theorem (CRT).

Unit 2: Symmetric Key Cryptography

Stream ciphers: RC4, ChaCha20. Block ciphers: DES, 3DES, AES (Rijndael). Modes of operation: ECB, CBC, CTR, GCM. Padding schemes. Cryptanalysis: differential and linear attacks.

Unit 3: Public Key Cryptography

RSA: key generation, encryption, decryption, attacks. Diffie-Hellman key exchange. ElGamal. Elliptic Curve Cryptography (ECC): ECDH, ECDSA. Certificates and PKI.

Unit 4: Cryptographic Hash Functions & Protocols

MD5, SHA-1, SHA-2, SHA-3 (Keccak). HMAC, digital signatures (RSA-PSS, DSA). Zero-knowledge proofs. Commitment schemes. Secure multi-party computation.

Unit 5: Post-Quantum Cryptography

Quantum computing threat model (Shor's algorithm, Grover's algorithm). Lattice-based cryptography (NTRU, Kyber). Code-based cryptography (McEliece). NIST PQC standardization. Blockchain and cryptocurrency cryptography.

Total Hours: 45

Textbooks
  • "Introduction to Modern Cryptography" by Katz & Lindell
  • "Cryptography and Network Security" by William Stallings
Reference Books
  • "Applied Cryptography" by Bruce Schneier
  • "Serious Cryptography" by Jean-Philippe Aumasson

CS5202: Generative AI & Large Language Models

L-T-P-C: 3-0-0-3 | Credits: 3 | Faculty: Dr. Nidhi

Prerequisites: Deep Neural Networks, Linear Algebra, Python

Unit 1: Transformer Architecture Deep Dive

Self-attention mechanism, multi-head attention. Positional encodings. Encoder-decoder vs decoder-only architectures. Layer normalization, feed-forward sublayers. Scaling laws for language models.

Unit 2: Pre-training & Fine-tuning LLMs

Masked Language Modeling (BERT). Causal Language Modeling (GPT family). Next-sentence prediction. Instruction tuning, RLHF (Reinforcement Learning from Human Feedback). LoRA and parameter-efficient fine-tuning.

Unit 3: Generative Models

Variational Autoencoders (VAEs): reparameterization trick, ELBO. Generative Adversarial Networks (GANs): mode collapse, training instability. Diffusion models: DDPM, score matching. Image generation with Stable Diffusion.

Unit 4: Prompt Engineering & RAG

Zero-shot, few-shot, chain-of-thought prompting. Retrieval Augmented Generation (RAG) pipeline. Vector databases (Pinecone, ChromaDB). LangChain and LlamaIndex frameworks. Agents and tool use.

Unit 5: Ethics, Safety & Deployment

Hallucination and factual grounding. Bias and fairness in LLMs. Constitutional AI and alignment. Model evaluation: BLEU, ROUGE, BERTScore, human eval. Deployment: quantization, ONNX, API design.

Total Hours: 42
